# file-vec
Store a Vec like data structure on the disk instead of in RAM. This could be useful if you need slice-like access to a large enough dataset to not fit comfortably in memory, or persistent storage of a cache of Plain Old Data. If you need actual database operations consider using a more proper database library instead. file-vec is more intended for very large amounts of simple data that are infeasible or unperformant to read into RAM or recompute when needed.
